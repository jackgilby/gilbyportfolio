Homecoming Site

Here's a website that was made for UW-Stout 2021 Homecoming, which was themed 'Press Stout to Play'. It features a pixelated visual language and utilizes Javascript to render an animated star-like background utilizing the branding color palette.

It was primarily used to show a list of events as well as link to the event details on a seperate website, as well as show clues for a Medallion Hunt event and post team scores.
